# 02 Features & Functionality - IS Helper

## บทบรรยาย: การออกแบบประสบการณ์การใช้งาน (User Journey)
เอกสารฉบับนี้มีหน้าที่อธิบาย "ความสามารถ" ของตัวแอปพลิเคชันอย่างละเอียด โดยเน้นการบรรยายว่าแต่ละฟีเจอร์ช่วยตอบโจทย์ผู้ใช้งานในสถานการณ์จริงได้อย่างไร ข้อมูลในนี้จะช่วยให้เห็นภาพรวมของ Workflow ตั้งแต่การเริ่มต้นหัวข้อวิจัย ไปจนถึงการสำเร็จงานเป็นรูปเล่ม PDF


## 1. ฟีเจอร์สำหรับนักเรียน (Student Operations)
ระบบถูกออกแบบมาเป็นลำดับขั้นตอน (Workflow-based) เพื่อไม่ให้นักเรียนรู้สึกว่างานเยอะเกินไป:
- **Project Stepper:** ระบบนำทาง 5 บท (Chapter 1-5) ซึ่งในแต่ละบทจะมีการซอยย่อยเป็นหัวข้อเล็กๆ เช่น ความเป็นมา, วัตถุประสงค์, ขอบเขต เป็นต้น
- **AI Drafter:** ปุ่มช่วยเหลือในทุกหัวข้อ เมื่อกดแล้ว AI จะนำข้อมูลพื้นฐานที่นักเรียนป้อนไว้มาขยายความและเรียบเรียงให้
- **Live Preview:** แสดงหน้าตาของรายงานที่จะออกมาในรูปแบบ PDF แบบ Real-time
- **Learning Resources:** ลิงก์ไปยังบทเรียนหรือตัวอย่างงานวิจัยที่ถูกต้องในแต่ละหัวข้อ
- **Onboarding System:** ระบบนำชมแอปครั้งแรก เพื่อสอนวิธีใช้เครื่องมืออย่างถูกต้อง

## 2. ฟีเจอร์สำหรับคุณครู (Teacher Dashboard)
- **Classroom Creation:** ครูสามารถสร้างห้องเรียน (เช่น "IS ม.5/1") และระบบจะเจนเนอเรต 6-character code เพื่อให้นักเรียนนำไปใส่
- **Visual Analytics:** กราฟแสดงจำนวนนักเรียนที่อยู่ระดับต่างๆ (กำลังเริ่ม, ทำถึงบท 3, เสร็จแล้ว)
- **Direct Feedback:** คุณครูสามารถเข้าไปดูหัวข้อที่นักเรียนเขียน และส่งคำแนะนำกลับไปได้
- **Resource Management:** คุณครูสามารถอัปโหลดไฟล์ตัวอย่างหรือคู่มือเฉพาะของโรงเรียนให้เด็กๆ ได้

## 3. ฟีเจอร์ส่วนกลางและ AI (Global Features)
- **Floating AI Assistant:** แชทบอทที่รู้จักบริบทของโปรเจกต์นักเรียน สามารถถามได้ว่า "ในบทนี้ใส่รูปภาพได้ไหม?" หรือ "ต้องเขียนอ้างอิงแบบไหน?"
- **Triple-Access Model:** ระบบที่รองรับทั้งผู้ใช้สายฟรี, สายที่มี API Key เอง, และสายเปย์ (Premium)
- **Theme Customization:** รองรับทั้ง Light Mode และ Galaxy (Dark) Mode เพื่อประสบการณ์การใช้งานที่ดีเยี่ยม
