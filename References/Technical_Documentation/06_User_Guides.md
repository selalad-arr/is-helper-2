# 06 User Guides - IS Helper

## บทบรรยาย: ก้าวแรกสู่ความสำเร็จในการทำโครงงาน
เอกสารนี้เปรียบเสมือน "แผ่นพับนำทาง" สำหรับผู้ใช้งานทุกระดับ ตั้งแต่นักเรียนที่เพิ่งเริ่มต้น คุณครูที่ต้องการตรวจงาน ไปจนถึงผู้ดูแลระบบที่ต้องบริหารหลังบ้าน เพื่อให้ทุกคนสามารถใช้เครื่องมือได้อย่างเต็มประสิทธิภาพโดยไม่สับสน


## สำหรับนักเรียน (Fast Start)
1. **เข้าสู่ระบบ:** ใช้ Google Login เพื่อความรวดเร็ว
2. **เลือกบทบาท:** เลือกเป็น "นักเรียน"
3. **เข้าร่วมห้องเรียน (ถ้ามี):** ใส่รหัสที่คุณครูให้มา เพื่อส่งงานผ่านระบบ
4. **เริ่มโปรเจกต์:** กดปุ่มบวกเพื่อใส่ชื่อเรื่อง
5. **ทำทีละบท:** เข้าไปในโปรเจกต์ แล้วทำตามหัวข้อย่อย หากนึกไม่ออก ให้กดปุ่ม "วิเคราะห์ด้วย AI" เพื่อร่างเนื้อหาเบื้องต้นให้
6. **ดาวน์โหลด:** เมื่อเสร็จทุกบท ให้ไปที่เมนู "รวมเล่มและดาวน์โหลด" เพื่อรับไฟล์ PDF

## สำหรับคุณครู
1. **สร้างแผงควบคุม:** เลือกบทบาทเป็น "คุณครู"
2. **สร้างห้องเรียน:** เมื่อสร้างเสร็จจะได้รหัส 6 หลัก ให้ส่งรหัสนี้ในกลุ่มแชทนักเรียน
3. **ติดตามงาน:** คลิกที่ห้องเรียนเพื่อดูโปรเจกต์ของเด็กๆ ทั้งหมด
4. **ให้คะแนน/คำแนะนำ:** สามารถพิมพ์คอมเมนต์กลับไปให้นักเรียนแก้ไขได้ในแอป

## สำหรับผู้ดูแล (Admin)
1. เข้าไปที่เมนู "Founder Dashboard" (เฉพาะ Admin เท่านั้น)
2. จัดการคำสั่งซื้อ Premium ที่รอการยืนยัน
3. ตรวจสอบสุขภาพของระบบและจำนวนการใช้งาน AI รายวัน
