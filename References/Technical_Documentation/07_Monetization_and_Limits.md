# 07 Monetization & Limits - IS Helper

## บทบรรยาย: ความยั่งยืนและการเข้าถึงอย่างเท่าเทียม
เอกสารสุดท้ายนี้จะอธิบายถึง "ทัศนคติ" ทางธุรกิจของโปรเจกต์ ว่าทำไมเราจึงต้องมีการจำกัดโควตาและมีระบบ Premium โดยมีเป้าหมายหลักไม่ใช่กำไรสูงสุด แต่คือความยั่งยืนของเซิร์ฟเวอร์เพื่อให้ระบบสามารถให้บริการนักเรียนไทยได้ในระยะยาว


## โมเดลธุรกิจและการสร้างรายได้
IS Helper ถูกบริหารจัดการโดยเน้นความยั่งยืน ไม่แสวงผลกำไรมหาศาล (Social Enterprise Model)

### ระบบ Premium (79 THB)
- **สิทธิประโยชน์:**
    - ปลดล็อก AI ไม่จำกัดครั้ง
    - สร้างโปรเจกต์ได้มากกว่า 3 โปรเจกต์
    - ดาวน์โหลดรายงานโดยไม่มีลายน้ำ (ถ้ามีในอนาคต)
    - ได้รับสัปดาห์แรกของฟีเจอร์ใหม่ก่อนใคร
- **การจัดการ:** เป็นระบบจ่ายเงินครั้งเดียว (Life-time or Semester-based) เพื่อให้เข้ากับงบประมาณนักเรียน

### ข้อจำกัดทางเทคนิค (Usage Quotas)
- **Free Account:** 
    - 3 AI Requests / Day
    - 3 Projects Max
- **Premium Account:**
    - Unlimited AI Requests
    - Unlimited Projects
- **Custom API Key:**
    - Unlimited AI Requests (Based on their own Google Cloud quota)

## ทำไมต้องมีข้อจำกัด?
เพื่อป้องกันการแสวงหาผลประโยชน์จากการใช้ API ของแอปพลิเคชัน (Bot abuse) และเพื่อให้ระบบมีทรัพยากรที่เพียงพอสำหรับนักเรียนจริงที่ต้องการความช่วยเหลือในช่วงฤดูทำโครงงาน
