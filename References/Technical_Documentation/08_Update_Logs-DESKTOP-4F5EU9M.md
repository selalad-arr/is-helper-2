# 08 Update Logs - IS Helper (Changelog)

## บทบรรยาย: บันทึกการเจริญเติบโตของระบบ
ไฟล์นี้มีหน้าที่เก็บรวบรวมประวัติการเปลี่ยนแปลง (Change Log) ของ Web Application ทั้งหมด ไม่ว่าจะเป็นการเพิ่มฟีเจอร์ใหม่ การแก้ไขบัค หรือการปรับปรุงประสิทธิภาพ เพื่อให้ทีมพัฒนาและผู้เกี่ยวข้องสามารถติดตามพัฒนาการของระบบได้ตามลำดับเวลา

---

## [2026-04-11] - ปรับปรุงรากฐานและระบบบริหารจัดการ
### เพิ่มเติม (Added)
- **Founder Dashboard:** ระบบจัดการหลังบ้านสำหรับ Admin (selalad@gmail.com)
- **Triple-Tier AI Access:** ระบบแบ่งระดับการเข้าถึง AI (Free, API Key, Premium)
- **Technical Documentation Suite:** ระบบเอกสารคู่มือและ Logbook อย่างละเอียด (00-08)
- **Galaxy Theme:** ปรับปรุงโหมดมืด (Dark Mode) ให้เป็นธีมอวกาศที่สวยงามระดับ Premium

### ปรับปรุง (Improved)
- **PDF Export Engine:** พัฒนาการจัดรูปแบบหน้ากระดาษ A4 ให้รองรับภาษาไทยได้แม่นยำขึ้น
- **UI/UX:** อัปเกรดเป็น Tailwind CSS 4 เพื่อประสิทธิภาพที่สูงขึ้น
- **Student Onboarding:** ปรับปรุงกระบวนการเริ่มต้นใช้งานสำหรับนักเรียนใหม่

### แก้ไข (Fixed)
- แก้ไขปัญหา Loading Stall ในหน้าเริ่มต้นด้วยระบบ Force-load Timeout
- แก้ไขปัญหาการเชื่อมต่อ Database สำหรับรายชื่อนักเรียนในห้องเรียน

## [2026-04-15] - Hybrid Journey System & UI Modernization
### เพิ่มเติม (Added)
- **Project Dashboard Hub (Non-linear):** ระบบแผงควบคุม 4 โซนย่อยที่ให้นักเรียนเลือกทำขั้นตอนใดก่อนก็ได้
- **Scenario-based Onboarding:** ระบบ AI ถามสถานการณ์ (The Lost, The Maker, The Perfectionist, The Rusher) และวาร์ปไปยังขั้นตอนที่เหมาะสมทันที
- **Flexible Data Flow:** ระบบจัดเก็บข้อมูลแบบกระจายศูนย์ที่สามารถรวบรวมเข้าเอกสารสมบูรณ์ได้แม้ข้ามขั้นตอน
- **Smart Report Fallback:** ระบบสร้างหัวข้อรายงานอัตโนมัติสำหรับขั้นตอนที่ข้ามไป เพื่อให้การพิมพ์รายงานทำได้ทันที

### ปรับปรุง (Improved)
- **UI Architecture:** ปรับโครงสร้าง `ISPage.tsx` ให้เชื่อมต่อกับ `ProjectHub` โดยอัตโนมัติเมื่อเป็นคอร์สสร้างโครงงาน
- **Visual Feedback:** เพิ่ม micro-animations และความงามระดับ Premium ในหน้าสร้างโครงงาน

### แก้ไข (Fixed)
- ตรวจสอบและยืนยันระบบ Save ข้อมูล (Firestore Merge) ครบทุกจุด ป้องกันข้อมูลหาย
- แก้ไขปัญหา Icon แสดงผลผิดพลาดใน Dashboard ตัวใหม่
